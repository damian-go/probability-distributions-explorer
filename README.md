# Probability Distributions Explorer

An interactive Shiny app for exploring and visualizing various probability distributions and their simulations.

## Features

- Visualize theoretical probability distributions.
- Adjust distribution parameters dynamically.
- Perform simulations with a specified sample size.
- Compare empirical results with theoretical distributions.
- Supports Normal, Binomial, Poisson, Exponential, and Uniform distributions.
